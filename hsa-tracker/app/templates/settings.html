Setup Instructions

To enable Outlook email sync, you need to register an app in Azure AD:

  1. Go to Azure AD Portal: Azure App Registrations
  2. Create a new registration:
    • Name: "HSA Tracker"
    • Supported account types: "Personal Microsoft accounts only" (for personal Outlook)
    • Redirect URI: Leave blank (we use device code flow)
  3. Configure API permissions:
    • Microsoft Graph > Delegated permissions
    • Add: Mail.Read, User.Read
  4. Enable public client flows:
    • Go to Authentication
    • Enable "Allow public client flows"
  5. Copy credentials:
    • Application (client) ID
    • Directory (tenant) ID (use "consumers" for personal accounts)

Environment Configuration

Create a .env file in the hsa-tracker directory with:

AZURE_CLIENT_ID=your-client-id
AZURE_CLIENT_SECRET=optional-for-public-client
AZURE_TENANT_ID=consumers

FLASK_SECRET_KEY=generate-a-random-key
FLASK_DEBUG=True

About

HSA Spending Tracker

Track your Health Savings Account expenses by automatically scanning your email for HSA-related receipts and documents.

Features:

  • Automatic email scanning for HSA transactions
  • Document extraction and OCR for receipts
  • Spending categorization (Medical, Dental, Vision, Prescription)
  • Transaction verification and reimbursement tracking
